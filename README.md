# HTML5 Video Player
This is a custom HTML5 based video player featuring keyboard shortcuts and support for <b>picture-in-picture</b>, built with simple HTML, CSS, JS and lots of ♥.

#### Supported Keyboard Shortcuts 
1. You can press <kbd>space</kbd> to play or pause the video.
1. You can press <kbd>f</kbd> to toggle fullscreen.
1. You can press <kbd>p</kbd> to toggle between picture-in-picture mode.
1. You can press <kbd>m</kbd> to mute or unmute the audio.
